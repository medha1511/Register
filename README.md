# registration-dapp
Registration Page that suits for any dapp built on ganache local ethereum blockchain.

# Requirements
1. Ganache Blockchain Network
2. Metamask
3. Truffle Suite
4. npm 

# Usage
Kindly make sure you have installed all the above said requirements. 

# Commands
truffle compile <br/>
truffle migrate <br/>
npm run dev <br/>
